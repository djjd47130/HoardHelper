Background

Rewind to several years ago, and an online friend of mine is commissioning me to create a few small tools to help him perform bulk operations on his media libraries. Over time, those tools have become scattered, messy, fragmented, and more is being expected lately.

Back to current times - I was recently itching for an interesting project where I could utilize DWScript. After a somewhat successful (yet meaningless) implementation for converting units-of-measurement, I decided to start this new project to replace and enhance all those tools I have made over time.

 

GitHub Repository

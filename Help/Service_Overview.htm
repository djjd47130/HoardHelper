Hoard Helper Service Overview

A part of the Hoard Helper application is an optional service which can serve a number of additional features. It gets installed into Windows when you run the software installer. If you do not already have this service installed, run the installer and be sure to select the Hoard Helper Service component.

 

Metadata Indexing

Creating and maintaining an index of all media metadata is critical to the performance and flexibility of the Hoard Helper system. All embedded metadata tags found in file headers are automatically extracted and saved in a central database.

Auto Script Execution

You may have certain scripts that you'd like to execute on a regular basis.This service is responsible for executing your scripts per your defined criteria. This is useful for general maintenance, such as deleting unnecessary or undesired files, or removing empty directories.

More Info

 

Media Organization

Automatically arranging your hoard of data into a meaningful folder structure is a major priority of the Hoard Helper system. The service is responsible for copying (or moving) files from your source "unsorted" media directory into the clean organized structure of your liking.

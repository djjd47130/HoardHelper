DWScript Syntax

The Hoard Helper system uses DWScript (Delphi Web Script) for its scripting language. This is primarily Pascal based, and supports pretty much all basic Pascal / Delphi syntax, with several additions.

The DWScript syntax is too large to document here, but as an example, you can implement things like:

  • Arrays / lists
  • Loops
  • Functions
  • Classes / Objects
  • Enumerations
  • Records
  • Constants
  • Conditionals
  • try/finally and try/except blocks
  • And much more

Please refer to the official documentation of DWScript for details.

 

Scripted Types

Certain data types are hard-coded into the scripting language, such as:

  • String - Text data (array of characters).
  • Integer - Whole number (no decimals).
  • Float - Decimal number.
  • Boolean - True or False.
  • etc.

Scripted Functions

There are many functions hard-coded into the scripting language, such as:

  • GetFiles - Returns an array of filenames matching search criteria.
  • ExtractFilePath - Returns a string containing the path where a file is located.
  • CreateDir - Creates a new directory.
  • DeleteFile - Deletes a given file.
  • GetTags - Returns array containing name/value pairs of file tags / metadata.
  • GetFileSize - Returns the size of a given file.
  • etc.

Library Paths

The Hoard Helper scripting language does not allow you to reference file paths directly in your filesystem. Instead, you must reference files by using the name of one of your registered media libraries.

For example, instead of referencing "D:\Media\Videos\Movies\SomeMovie.MP4", you would access it more like "Movies\SomeMovie.MP4 ".

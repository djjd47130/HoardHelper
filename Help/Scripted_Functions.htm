The following functions are available within the scripting language:

Hoard Helper Utils:

  • function GetTickCount : Integer;
    • Returns number of milliseconds of CPU runtime - useful for measuring time spent on scripted operations.
  • procedure PrintLn (S: String);
    • Prints a line of text into the output control.
    • Param S - The string to be printed into the output control.
  • procedure BeginPrintLn ;
    • Triggers output control to pause rendering until EndPrintLn is called.
  • procedure EndPrintLn ;
    • Triggers output control to resume rendering after BeginPrintLn was called.

String Utils:

  • function Copy (S: String; Index: Integer; Count: Integer): String;
    • Copies a selection of text from a string.
    • Param S - The string containing the text to be copied.
    • Param Index - The starting character index within the string to copy.
    • Param Count - The number of chracters to copy from Index .
  • function Pos (SubStr: String; Str: String; Offset: Integer = 1): Integer;
    • Finds the position of text within a string.
    • Param SubStr - The string you're looking for.
    • Param Str - The larger string you're searching in.
    • Param Offset- The number of characters to offset the search from. 
  • function UpperCase (Str: String): String;
    • Returns a string in UPPER CASE.
    • Param Str - The string to be converted to upper case.
  • function LowerCase (Str: String): String;
    • Returns a string in lower case. 
    • Param Str - The string to be converted to lower case.
  • procedure Delete (var S: String; Index: Integer; Count: Integer);
    • Deletes a section of text from a string.
    • Param S - The string containing the text to be deleted.
    • Param Index - The starting character index within the string to delete from.
    • Param Count - The number of chracters to delete from Index .
  • function sLineBreak : String;
    • Returns ANSII characters for a line break (#10 #13).
  • function sTab : String;
    • Returns ANSII character for a tab (#9).
  • function SplitString (S: String; Delimiter: String): Array of String;
    • Returns an array of string containing elements of a given string split by a delimiter.
    • Param S - The string to be split.
    • Param Delimiter - The delimiter to use to split S into an array of Strings.

File Utils:

  • function FileExists (Filename: String): Boolean;
    • Returns whether a given filename exists.
  • function DirExists (Filename: String): Boolean;
    • Returns whether a given directory name exists.
  • function GetFiles (Dir: String; SearchPattern: String = '*.*'; Recursive: Boolean = False): Array of String;
    • Returns an array containing filenames matching a file search.
  • function GetDirs (Dir: String; SearchPattern: String = '*'; Recursive: Boolean = False): Array of String;
    • Returns an array containing directories matching a directory search.
  • function ExtractFileName (Filename: String): String;
    • Returns a string containing only the filename portion of a full path / filename with its extension.
  • function ExtractFileNameWithoutExt (Filename: String): String;
    • Returns a string containing only the filename portion of a full path / filename without its extension.
  • function ExtractFilePath (Filename: String): String;
    • Returns a string containing only the path portion of a full path / filename.
  • function ExtractFileExt (Filename: String): String;
    • Returns a string containing only the last extension found in a filename including the dot.
  • function PathCombine (P1: String; P2: String): String;
    • Returns a string combining two path strings into one using appropriate path delimiter automatically.
  • function GetParentDir (Path: String): String;
    • Returns a string containing the parent directory of a given directory.
  • function GetGenericFileType (Ext: String): String;
    • Returns a string describing the file association of a given file extension.
    • Param Ext - The file extension to be looked up (including the '.').
  • function FileInUse (Filename: String): Boolean;
    • Returns whether or not a file is currently in use (locked from editing).
  • procedure OpenFile (Filename: String);
    • Opens a given file (or folder) in the default application associated with it.

  • function CopyFile (Source: String; Dest: String; Overwrite: Boolean = False): Boolean; < UL >

  • function DeleteFile (Filename: Boolean): Boolean; < UL >
    • Deletes a file.
  • function CreateDir (Path: String): Boolean;
    • Creates a new directory.
  • function MoveDir (Source: String; Dest: String; Overwrite: Boolean = False): Boolean;
    • Movies a directory and its contents from a source path to a destination path.
  • function CopyDir (Source: String; Dest: String; Overwrite: Boolean = False): Boolean;
    • Copies a directory and its contents from a source path to a destination path.< /UL >< /UL >
  • function DeleteDir (Dir: String): Boolean; < UL >
    • Deletes a directory and its contents.
  • function GetFileSize (Filename: String): Integer;
    • Returns the size of a given file in Bytes.
  • function GetFileSizeStr (Filename: String): String;
    • Returns a user-friendly string containing the size of a given file.
  • function GetFileDateModified (Filename: String): String;
    • Returns the date modified of a given file.
  • function SaveTextFile(Filename: String; Data: Array of String; Overwrite: Boolean = Falsae): Boolean;
    • Saves a string array to a given file.
    • Param Filename - The filename to save text to.
    • Param Data - The string array to be written to the file.
    • Param Overwrite - Whether to overwrite an existing file, if applicable.

Tags Utils:  

  • function ExtractID3v1 (Filename: String): Array of String;
    • Returns an array of string containing name/value pairs of the ID3v1 tags within a given audio file.
  • function GetTags (Filename: String): Array of String;
    • Returns an array of string containing name/value pairs of all tags / metadata found within a given file.
